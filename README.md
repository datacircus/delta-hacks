# delta-hacks
This is a collection of neat Delta Lake Hacks. Originally inspired by the Delta Lake webinar called "Delta Hacks: How Delta can propel organizations to AI data readiness"

## Getting Started

If you don't have pyenv installed, please do so. It is a time saver.

~~~bash
brew install pyenv
~~~

Now, if you're feeling like just having **claude** do the install work for you. 

Use the following prompt.
~~~prompt
Please install the newest version of python using pyenv. pyenv is already in my PATH.
~~~

then 

~~~prompt
go ahead and set the global version to 3.14.3 and confirm it is correctly set.
~~~
